Integritetspolicy

Senast uppdaterad: mars 2026

Studievägsguiden värnar om din personliga integritet. Den här policyn förklarar vilka uppgifter vi samlar in, varför vi gör det och hur du kan kontakta oss.

Vem ansvarar för dina uppgifter?

Studievägsguiden är personuppgiftsansvarig för de uppgifter som samlas in via studievagsguiden.se. Kontakta oss på [email protected] vid frågor.

Vilka uppgifter samlar vi in?

  • Namn — för att personalisera din plan
  • Telefonnummer — för att kunna kontakta dig om din plan
  • E-postadress — för att spara din plan och skicka information
  • Svar i formuläret — utbildningspreferenser, ekonomisk situation och liknande — för att generera din personliga studieplan

Varför samlar vi in uppgifterna?

  • För att skapa och spara din personliga studieplan
  • För att du ska kunna logga in och se din plan igen
  • För att kunna kontakta dig med information om din studieväg

Rättslig grund: samtycke — du godkänner detta i formuläret innan vi sparar dina uppgifter.

Hur länge sparar vi uppgifterna?

Vi sparar dina uppgifter så länge du har ett aktivt konto eller tills du begär att de raderas. Kontakta oss på [email protected] för att begära radering.

Delas uppgifterna med någon?

Vi delar inte dina personuppgifter med tredje part förutom vår databasplattform Supabase (EU-server, Frankfurt) som används för att lagra din plan säkert.

Dina rättigheter

Du har rätt att:

  • Få ut de uppgifter vi har om dig (registerutdrag)
  • Begära rättelse av felaktiga uppgifter
  • Radera ditt konto och all din data — direkt i appen via "Radera mitt konto" när du är inloggad på studievagsguiden.se/min-plan
  • Återkalla ditt samtycke när som helst
